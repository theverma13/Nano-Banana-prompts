# I. Strategic & Architectural Thinking

1. **The "Strategy North Star" Map**

```
Prompt for a high-concept diagram where your core business values are the "roots" of a tree and the market outcomes are the "fruit."
```

2. **Architecture of a Philosophy**

```
Visualize a philosopher's worldview (e.g., Stoicism) as a structural floor plan—each room representing a core tenet like Amor Fati or Memento Mori.
```

3. **The Recursive Loop Diagram**

```
Map out "vicious cycles" in your life or business where the text labels explain the transition from one phase of the failure loop to the next.
```

4. **A "Tech Stack" Totem**

```
Instead of a boring list, generate a vertical, intricate totem pole where each carving represents a layer of your software or hardware stack.
```

5. **Organizational "Solar Systems"**

```
Model your company hierarchy where the CEO is the sun and departments are planets, with text labels indicating the "gravity" (influence) each has on the other.
```

6. **Product Genealogy Trees**

```
Trace the "evolution" of an idea from its 19th-century roots to a modern AI application, rendered as a biological evolution chart.
```

7. **Ethical Boundary Maps**

```
Draw a "topographic map" where mountains represent high ethical risks and valleys represent safe zones for a specific project.
```

8. **The "Decision Forest"**

```
A flowchart where each decision path is a literal trail through a dense forest, showing where you might get "lost" without specific data.
```

9. **Vulnerability Blueprints**

```
A literal blueprint of a fortress where the "walls" are your strengths and the "cracks" (labeled with text) are your identified weaknesses.
```

10. **Global Trend "Weather Maps"**

```
Visualize market trends as high-pressure and low-pressure systems over a literal map of your industry.
```

# II. Creative & Narrative Exploration

1. **Emotional Periodic Tables**

```
Create a "Periodic Table of Elements" for a specific emotion (e.g., Grief or Joy), with each "element" having its own symbol and atomic weight (intensity).
```

2. **Character "Psych-Sheets"**

```
A medical-style X-ray of a fictional character where the labels point to different parts of their anatomy to explain their trauma and motivations.
```

3. **Magic System Infographics**

```
For writers, a diagram showing the "laws of physics" for a magic system, including the "energy cost" of spells rendered in text.
```

4. **Concept Mashup "Anatomy"**

```
Ask for a cross-section of a "Steampunk Smartwatch," showing how 19th-century gears would theoretically interact with digital code.
```

5. **Cinematic Tone-Maps**

```
A horizontal "EKG" line where the spikes represent tension levels in a movie script, with text annotations for key plot points.
```

6. **History of a Non-Existent Object**

```
A multi-panel storyboard showing the "archaeology" of a futuristic relic.
```

7. **Literary Constellations**

```
Visualize a complex novel (like Dune) where the characters are stars and the lines between them are labeled with their relationship types (Betrayal, Love, Fealty).
```

8. **The "Museum of Failed Ideas"**

```
A 3D isometric gallery view of "exhibits," each representing a project you abandoned, with "plaques" explaining why.
```

9. **Dream Cartography**

```
Describe a recurring dream and ask the AI to map it as a 17th-century nautical chart.
```

10. **Visualizing Music**

```
Describe a song's structure and ask for it to be rendered as an architectural cathedral where the "pillars" are the bassline and the "stained glass" is the melody.
```

# III. Professional & Knowledge Management

```
A diagram of everything you've learned and forgotten, mapped like an archaeological dig site.
```

2. **The "Skill Constellation"**

```
A map of your professional skills where the distance between nodes represents how "transferable" they are to one another.
```

3. **AI Workflow "Circuitry"**

```
Visualize your daily AI usage as a literal circuit board, showing where the "input" enters and where the "final value" is output.
```

4. **A "Taxonomy of Focus"**

```
A biological classification chart (Kingdom, Phylum, etc.) for different types of deep work and shallow work.
```

5. **The "Impulse Purchase" Funnel**

```
A diagram of the psychological triggers that lead to a purchase, rendered as a Rube Goldberg machine.
```

6. **Corporate Culture "Icebergs"**

```
A diagram showing the "visible" parts of a company (logos, slogans) above water and the "invisible" parts (office politics, unspoken rules) below.
```

7. **The "Burnout Dashboard"**

```
A conceptual car dashboard where the "gauges" represent your current levels of sleep, stress, and creativity.
```

8. **Learning "S-Curves"**

```
A visual representation of the "plateaus" and "leaps" in learning a new language, with text milestones.
```

9. **Project "Autopsy" Reports**

```
A visual diagram of a failed project where the "cause of death" is identified at different structural points.
```

10. **Mental Model "Swiss Army Knives"**

```
A visual of a literal multi-tool where each blade is labeled with a mental model (e.g., First Principles, Occam's Razor).
```

# IV. Deeply Philosophical & Abstract

1. **The "Architecture of Time"**

```
A 3D diagram showing how different cultures perceive time (linear vs. cyclical vs. chaotic).
```

2. **Visualizing Paradoxes**

```
Ask for a diagram explaining the "Ship of Theseus" or the "Grandfather Paradox" using literal ship or clock parts with text labels.
```

3. **The "Shadow Self" Venn Diagram**

```
A diagram showing the intersection between who you present to the world and your "Jungian Shadow."
```

4. **Complexity vs. Simplicity Scales**

```
A visual balance scale where "Modern Life" is on one side and "Essential Needs" is on the other, with labels for what weighs each down.
```

5. **The "Soul's Ecosystem"**

```
A biological diagram of a pond where different organisms represent your traits (e.g., "The Frog of Anxiety," "The Lilypad of Peace").
```

6. **Ethics of AI Flowchart**

```
A path leading to a fork in the road—one labeled "Utopia" and one "Dystopia"—with the critical decision points highlighted.
```

7. **The "Identity Mosaic"**

```
A portrait made of smaller, labeled icons representing every person and experience that shaped you.
```

8. **Map of the "Unspeakable"**

```
A map of a territory where the landmarks are things words cannot describe, using visual metaphors like "The Sea of Silence."
```

9. **The "Ego-Centric" Solar System**

```
A diagram of a person's life where their Ego is the sun and their friends/family are moons revolving around it.
```

10. **Entropy vs. Order Visuals**

```
A split-screen diagram showing a city at its peak and the same city 1,000 years later, with text explaining the transition.
```

A physics-style diagram showing the "Gravity of Virality" or the "Friction of Disagreement."
